# internship-seatbookingsystem

The HR can input the location, building, floor, department and no of seats to be assigned to that department 

The user can login and input the date or daterange, location and project they are working on which will then be checked in a table which has the details of the booked seats  

From the booked seat data the seat name, department id, location id, building, floor values of the data which matches the given date, location and user's department are checked with assigned seat data and seats which dont match the values are displayed which are the available seats 

If the seats in the user's department is full then other available seats are displayed  

On booking the seat, details of the seat booked by the user is displayed with the option to cancel the booking
